# SearchSwarm Project Page

Static project page for:

**SearchSwarm: Towards Delegation Intelligence in Agentic LLMs for Long-Horizon Deep Research**

## Local Preview

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 4173
```

Then visit `http://127.0.0.1:4173/`.

## Deploy

Upload the contents of this folder to a GitHub Pages repository. Keep `index.html` at the repository root and preserve the `static/images/` and `case_studies/` paths.

## Update Before Release

- Replace `Anonymous Authors` with final author and affiliation metadata.
- Replace placeholder `Paper`, `Model`, and `Dataset` links with public URLs.
- Update the BibTeX block after acceptance or arXiv release.
